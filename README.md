# atm-web-frontend
Writing a dockerfile to running an atm web app

## Installation
### Step 1: Clone this repository
```
https://github.com/jittaearn/atm-web-frontend.git
```
### Step 2: Go to folder
```
cd atm-web-frontend
```
### Step 3: Build docker
```
docker build -t atm-docker
```
### Step 4: Run docker
```
docker run -d -p 9000:8090 atm-docker
```
